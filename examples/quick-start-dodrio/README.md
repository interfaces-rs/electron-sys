# quick-start-dodrio

A simple `electron-sys` example that uses [dodrio](https://github.com/fitzgen/dodrio).

## Requirements

- [npm](https://nodejs.org/en/download/)
- [rustup](https://rustup.rs/)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/)

## Building

```
$ npm install
$ npm start
```
